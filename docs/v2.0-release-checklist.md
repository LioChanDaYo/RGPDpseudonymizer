## üö® Known Limitations (Ship with v2.0)

### UI-001: 200% DPI Scaling Layout Issues

**Impact:** Low - Minor widget overlapping/squishing at 200% system DPI scaling
**Affected Users:** Users with extreme accessibility needs (200%+ scaling)
**WCAG Compliance:** PASS - Text remains readable, all functionality accessible
**Mitigation:** Consider layout refinements in v2.1 based on user feedback
**Documented:** [stories/6.7.accessibility-wcag-aa.story.md](stories/6.7.accessibility-wcag-aa.story.md#L469-L475)

### DATA-001: Entity Type Count Query (Carryforward from v1.1)

**Impact:** Medium - Results screen shows cumulative database totals, not current document
**Affected Users:** Users with large mapping databases
**Status:** Known issue since Story 6.3, acceptable for v2.0
**Fix:** Filter to current-document entities or add per-type counts to core `ProcessingResult`
**Documented:** [backlog.md](backlog.md) (Open QA Items)
