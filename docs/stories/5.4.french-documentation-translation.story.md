### MkDocs i18n Plugin

**[Source: mkdocs.yml, pyproject.toml]**

The project uses MkDocs `^1.5.0` with `mkdocs-material ^9.5.0`. The recommended i18n approach is `mkdocs-static-i18n`, which supports:

- **Suffix-based localization:** `installation.md` (default/EN) + `installation.fr.md` (French) in same directory
- **Automatic language toggle:** Plugin adds language switcher to the MkDocs Material theme
- **Fallback behavior:** Pages without a `.fr.md` version fall back to the English version
- **No directory restructuring needed:** French files live alongside English files with `.fr.md` suffix

This is the standard approach for MkDocs Material. No custom code or complex configuration required.
