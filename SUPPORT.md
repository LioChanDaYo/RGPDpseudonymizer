# Support

## How to Get Help

### Self-Help Checklist

Before asking for help, please try these steps:

1. **Check the documentation:** [https://liochandayo.github.io/RGPDpseudonymizer/](https://liochandayo.github.io/RGPDpseudonymizer/)
2. **Read the FAQ:** [FAQ page](https://liochandayo.github.io/RGPDpseudonymizer/faq/)
3. **Check Troubleshooting:** [Troubleshooting guide](https://liochandayo.github.io/RGPDpseudonymizer/troubleshooting/)
4. **Search existing issues:** [GitHub Issues](https://github.com/LioChanDaYo/RGPDpseudonymizer/issues)
5. **Search discussions:** [GitHub Discussions](https://github.com/LioChanDaYo/RGPDpseudonymizer/discussions)

### Ask a Question

For general questions, usage help, or best practices:

- **GitHub Discussions:** [https://github.com/LioChanDaYo/RGPDpseudonymizer/discussions](https://github.com/LioChanDaYo/RGPDpseudonymizer/discussions)

### Report a Bug

If you've found a bug:

- **GitHub Issues:** [https://github.com/LioChanDaYo/RGPDpseudonymizer/issues](https://github.com/LioChanDaYo/RGPDpseudonymizer/issues)
- Use the Bug Report template and include: OS, Python version, steps to reproduce

### Request a Feature

Have an idea for an improvement?

- **GitHub Issues:** [https://github.com/LioChanDaYo/RGPDpseudonymizer/issues](https://github.com/LioChanDaYo/RGPDpseudonymizer/issues)
- Use the Feature Request template

### Security Issues

For security vulnerabilities, please see [SECURITY.md](SECURITY.md) for responsible disclosure instructions. Do **not** open a public issue for security vulnerabilities.
